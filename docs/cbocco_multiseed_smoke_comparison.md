- Date: 2026-06-30
- Executor: Codex
- Scope: Phase 7E integration smoke only; no optimizer behavior change.

## Purpose

Phase 7E repeats the Phase 7C CBOCC grouping-source smoke check over seeds `1, 3, 5`.
It compares the unchanged legacy `CBCCO` grouping path with an explicit-file grouping path generated by CWVIG capped-default inference on the first 10 dimensions, then completed to the 905D Flyki F1 dimension with singleton groups.

This phase is not a final optimization benchmark.

## Workflow

For each seed:

1. Generate a capped 10D CWVIG grouping with `configs/cwvig_presets/capped_default.json`.
2. Complete the grouping to 905D using singleton groups.
3. Validate the completed explicit `po/oo` files with `grouping_source_cli`.
4. Run legacy `CBCCO` through `legacy_by_function`.
5. Run completed-predicted `CBCCO` through `explicit_files`.
6. Archive stdout, stderr, exit code, result files, and grouping artifacts under `results/phase7e/seed_<seed>/`.
7. Analyze pairwise results with common-FE lookup using the last logged row with `FE <= common_fe`.

## Common-FE Policy

For completed seed pairs, the analyzer reports both:

- final-fitness comparison at each run's own final logged FE;
- common-FE comparison at `min(final_fe_legacy, final_fe_completed_predicted)`.

When the target FE is not exactly logged by a curve, the analyzer uses the last logged row with `FE <= common_fe`. This is conservative and avoids interpolation assumptions.

Observed Phase 7E final FE:

```text
seed 1 legacy=162000, completed_predicted=181600
seed 5 legacy=162000, completed_predicted=181800
```

Therefore Phase 7E is not FE-matched and must not be used as an optimization superiority claim.

Completed-predicted won the available seed pairs at final fitness and common-FE fitness, but those wins are smoke-test observations only because:

- completed-predicted uses 10D CWVIG plus singleton completion, not the final CWVIG-OSD method;
- original hard-overlap `CBOG_CBD` is unchanged;
- final-FE comparisons are not matched;
- seed 3 completed-predicted failed before producing a result file;
- no shared-variable-aware update policy exists in this phase.
