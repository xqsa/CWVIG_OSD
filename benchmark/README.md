# Benchmark Sources

This directory hosts external baseline code used by CWVIG-OSD.

- `flyki_overlap/`: local checkout of `Flyki/Large-Scale-Overlapping-Optimization`.
- `cec2013lsgo/`: local checkout of `dmolina/cec2013lsgo`.

The initial scaffold keeps each repository intact so their original build and
license metadata remain available. Integration wrappers should live outside
these directories unless a deliberate upstream patch is required.
